# GitHub Issues Tracker

## 📌 Project Overview

**GitHub Issues Tracker** is a beautifully designed web application that simulates GitHub's issue management system. 

It provides a clean, responsive grid interface to view, filter, and track project issues with essential details like priority, type, assignee, status, and creation date.

Built as a static frontend project, it demonstrates modern UI/UX practices with smooth visuals and intuitive navigation.

**Live Demo:**
https://git-hub-issue-tracker-a5.vercel.app/

---

## ✨ Key Features

- **Modern Responsive Grid** – Clean card-based layout
- **Priority & Type Labels** – Color-coded High, Medium, Low priorities and issue types (Bug, Enhancement, Documentation, Help Wanted)
- **Filtering Options** – All, Open, Closed tabs
- **Search Functionality** – Quickly find issues
- **Detailed Issue Cards** – Showing assignee, date, and status
- **Fully Responsive** – Works perfectly on mobile, tablet, and desktop
